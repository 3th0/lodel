Installation du fichier de configuration

Erreur: Le fichier lodelconfig.php à la racine de Lodel n'est pas lisible par le serveur web. Veuillez ajuster les droits.

Option 1

Une erreur est intervenue lors de la copie du fichier de configuration "lodelconfig.php" dans le répertoire racine de Lodel. Ceci est probablement dù au droits d'ecriture sur ces repertoires. Si vous souhaitez que le serveur fasse l'operation, il faut lui permettre d'écrire dans ce répertoire. Sinon, reportez vous à l'option 2.

Option 2

Le fichier de configuration est pret, il faut que vous l'installiez dans le repertoire racine de Lodel. Ce fichier doit être lisible par le serveur web, mais comme il contient des informations confidentielles (mot de passe), vous devez restreindre au maximum les droits de lecture sur ce fichier.

Pour installer le fichier de configuration, il y a trois possibilités:

  • Telecharger le fichier puis enregistrer le tel que à la racine de Lodel (par ftp par exemple).
    		puis
  • Visualiser le fichier, faites un copier coller dans un fichier texte nommé lodelconfig.php est enregistrer à la racine de Lodel. (veillez a ne laisser aucune ligne apres le ?> final):
  • Copier le fichier lodeladmin/ à la racine de Lodel.
