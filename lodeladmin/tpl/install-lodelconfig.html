Installation du fichier de configuration

Erreur: Le fichier lodelconfig.php à la racine de Lodel n'est pas lisible par le serveur web. Veuillez ajuster les droits.

[Option d'installation 1]

Une erreur est intervenue lors de la copie du fichier de configuration "lodelconfig.php" dans le répertoire racine de Lodel. Ceci est probablement dû aux droits d'écriture sur ces repertoires. Si vous souhaitez que le serveur fasse l'opération lui même, vous devez lui donner les droits d'écriture sur le répertoire racine de Lodel. Sinon, reportez vous à l'option 2 pour des solutions alternatives.

[Option d'installation 2]

Le fichier de configuration est prêt, il faut l'installer dans le répertoire racine de Lodel. Ce fichier doit être lisible par le serveur web, mais comme il contient des informations confidentielles (mot de passe), il faut restreindre au maximum les droits de lecture sur ce fichier.

Pour installer le fichier de configuration, il y a trois possibilités :

  • Télécharger le fichier puis l'enregistrer à la racine de Lodel (par ftp par exemple).
    		puis
  • Visualiser le fichier, faire un copier coller dans un fichier texte nommé lodelconfig.php et l'enregistrer ensuite à la racine de Lodel. (ne laissez aucune ligne apres le ?> final) :
  • Copier vous même le fichier lodeladmin/ à la racine de Lodel en le renomant 'lodelconfig.php'.
