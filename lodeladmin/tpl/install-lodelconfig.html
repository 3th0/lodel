Une erreur est intervenue lors de la copie du fichier de configuration "lodelconfig.php" dans le répertoire racine de Lodel. Ceci est probablement dù au droits d'ecriture sur ces repertoires. Si vous souhaitez que le serveur fasse l'operation, il faut lui permettre d'écrire dans ce répertoire. Cependant, pour plus de sécurité, il possible de gérer les droits d'accès par vous même et de façon stricte. Dans ce cas, faites la copie à la main du fichier lodeladmin/ (sans y porté aucune modifications) dans le repertoire racine de Lodel, et assurer les droits d'écriture et de lecture les plus restreints possibles (lecture pour le serveur, écriture pour personne sinon vous-même).

LODEL ne vient avec AUCUNE GARANTIE d'aucune sorte. Lisez le fichier LICENSE s'il vous plait.
