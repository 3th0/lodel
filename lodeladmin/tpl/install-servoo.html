Configuration du serveur de conversion de document ServOO

Le ServOO a répondu positivement:


La conversion de documents aux formats RTF, SXW ou DOC en xhtml est réalisée avec ServOO, un serveur basé sur OpenOffice.org et Writer2Latex

Deux possibilités:

  • Installer vous même un ServOO : cela nécessite l'installation de Open Office et Java sur votre serveur ! Cette installation requiert un serveur dedié et correctement protégé. Ce n'est que rarement possible chez les hebergeurs commerciaux (à moins de disposer d'un accès de type "serveur dédié" ou "serveur virtuel"). ServOO est distribué dans un paquetage séparé de Lodel sur http://sourceforge.net/projects/lodel/. Lorsque ServOO est installé, entrez les paramètres de connexion ci-dessous. Si vous installez un ServOO, nous vous encourageons à le mettre à disposition du public et à vous faire enregistrer sur http://www.lodel.org
  • Demander un compte sur un serveur ServOO publiquement accessible. La liste officielle des serveurs est disponible sur http://www.lodel.org. Entrez ensuite les paramètres de connexion ci-dessous.
