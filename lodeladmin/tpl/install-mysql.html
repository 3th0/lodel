Configuration de l'acces au serveur MySQL.

Option 1

Entrer les informations concernant le serveur MySQL. L'adresse du serveur est souvent disponible dans le support technique de l'hebergeur. Le nom d'utilisateur MySQL et le mot de passe sont personnels, et ont dù être fourni avec les autres informations concernant le compte ftp et les courriels. Attention, cependant, le nom d'utilisateur MySQL et le mot de passe sont souvent différents du nom d'utilisateur pour l'accès au site ou aux courriels.

Option 2

Erreur: Connection au serveur impossible. Vérifier les informations entrées. L'erreur produite est:
serveur MySQL
Utilisateur MySQL
Mot de passe de l'utilsateur MySQL
