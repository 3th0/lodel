Accès au serveur MySQL.

Renseignez les informations permettant à Lodel de se connecter à votre base de données.

[option 2]

Spécifiez les paramètres de connection sur le serveur MySQL. L'utilisateur MySQL doit avoir au minimum le droits d'USAGE sur le serveur. Pour créer un utilisateur, utiliser la commande GRANT dans mysql (ou phpMyAdmin). Par exemple:

GRANT USAGE ON *.* TO monutilisateur@monserveurweb IDENTIFIED BY 'monmotdepasse'; Dans votre cas:

GRANT USAGE ON *.* TO IDENTIFIED BY 'monmotdepasse';

Erreur: Connection au serveur impossible. Vérifiez les informations entrées. L'erreur produite est:


 

Adresse du serveur MySQL : elle a dû vous être notifiée dans la documentation technique de votre hébergeur. Exemple : localhost ou mysql.monhebergeur.com

Nom d'utilisateur MySQL et mot de passe : ils ont dû vous être communiqués par votre hébergeur.

En cas de doute : contactez votre service technique.
