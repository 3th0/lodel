Configuration de l'acces au serveur MySQL.

[option 1]

Entrez les informations concernant le serveur MySQL. L'adresse du serveur doit être indiquée dans la documentation technique de votre hébergeur (par exemple : mysql.monhebergeur.com, ou localhost, etc.) Le nom d'utilisateur MySQL et le mot de passe sont personnels : ils ont dû vous être communiqués en même temps que les autres informations concernant le compte ftp et les courriels. Attention cependant, le nom d'utilisateur MySQL et le mot de passe sont souvent différents du nom d'utilisateur pour l'accès au site ou aux courriels.
En cas de doute : contactez votre service technique pour plus d'informations.

[option 2]

Spécifiez les paramètres de connection sur le serveur MySQL. L'utilisateur MySQL doit avoir au minimum le droits d'USAGE sur le serveur. Pour créer un utilisateur, utiliser la commande GRANT dans mysql (ou phpMyAdmin). Par exemple:

GRANT USAGE ON *.* TO monutilisateur@monserveurweb IDENTIFIED BY 'monmotdepasse'; Dans votre cas:

GRANT USAGE ON *.* TO IDENTIFIED BY 'monmotdepasse';

Erreur: Connection au serveur impossible. Vérifiez les informations entrées. L'erreur produite est:

serveur MySQL
Utilisateur MySQL
Mot de passe de l'utilsateur MySQL
