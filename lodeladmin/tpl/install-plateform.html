Choix de la configuration de serveur.

Choisissez un type de configuration de serveur adaptée à votre hébergeur :

 Nom de la configurationCommentaire

Les configurations permettent de simplifier l'installation en l'adaptant à un hébergeur particulier. S'il n'y a pas de configuration adaptée a votre hébergeur, choisssez la configuration par défaut.

Si vous parvenez à installer Lodel sur un hébergeur qui n'est pas repertorié dans la liste sur http://www.lodel.org faites le savoir svp à lodel@lodel.org.
